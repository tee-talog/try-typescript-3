# try-typescript-3
Getting Started TypeScript!

# preparation
```
git clone https://github.com/tee-talog/try-typescript-3.git
cd try-typescript-3
npm ci
```

# npm scripts
* npm build
  * compile all .ts files
* npm start [filename]
  * run filename as TypeScript
* npm [format|fmt]
  * format .js and .ts files with Prettier
